Markdown

  • Tools and language to add formatting to simple documents
  • Readable and lightweight
  • Portable, convenient

Rmarkdown?

  • File format and package ecosystem to combine markdown-formatted text with R code
  • Compatible with HTML
  • Produces an intermediate .md file that can be processed to render multiple outputs (PDF, HTML, Word, etc.)

Quarto

  • Open Source technical and scientific publishing system
  • The new generation of rmarkdown
  • Multi-language
  • Multi-format
  • Well documented

YAML preamble

Contains document metadata to specify:

  • Visual options to be applied throughout

  • Author and date information

  • Output format

  • --- befor and after the preamble

  • Flimsy, be careful with indentations here

Example 1: Possums report

  • change theme (theme)
  • change code highlighting (highlight style)
  • hide or show code (code-fold, code-tools, code-summary)

Presentations

Quarto uses the reveal.js framework to create HTML presentations

  • Includes presenter tools

  • New slides delimited with --- or created with level two heading (##)

Example 2: Presentation about pets

  • Gradual reveal of lists and bullet points
  • Change slide background {background-color}
  • Export to PDF (E key toggles print preview)
